IEnumerable Cheatsheet

Filtering

Method Description Example
Where Filters a sequence of values based on a predicate. numbers.Where(n => n % 2 == 0)
Gets all even numbers from the sequence.

Sorting

Method Description Example
OrderBy Sorts the elements of a sequence in ascending order. numbers.OrderBy(n => n)
Sorts numbers in ascending order.
OrderByDescending Sorts the elements of a sequence in descending order. numbers.OrderByDescending(n => n)
Sorts numbers in descending order.
ThenBy Performs a secondary sort on the elements of a sequence in ascending order. persons.OrderBy(p => p.LastName).ThenBy(p => p.FirstName)
Sorts persons by last name, then by first name.
ThenByDescending Performs a secondary sort on the elements of a sequence in descending order. persons.OrderBy(p => p.LastName).ThenByDescending(p => p.FirstName)
Sorts persons by last name, then by first name in descending order.

Projection

Method Description Example
Select Projects each element of a sequence into a new form. numbers.Select(n => n * 2)
Doubles each number in the sequence.
SelectMany Projects each element of a sequence to an IEnumerable<T> and flattens the resulting sequences into one sequence. persons.SelectMany(p => p.PhoneNumbers)
Flattens phone numbers of persons.

Join Operations

Method Description Example
Join Correlates the elements of two sequences based on matching keys. students.Join(scores, s => s.ID, sc => sc.StudentID, (s, sc) => new { Name = s.Name, Score = sc.Score })
Joins students and scores based on student ID.
GroupJoin Correlates the elements of two sequences based on equality of keys and groups the results. categories.GroupJoin(products, c => c.ID, p => p.CategoryID, (c, prods) => new { Category = c.Name, Products = prods })
Groups products by category.

Set Operations

Method Description Example
Distinct Returns distinct elements from a sequence. numbers.Distinct()
Gets distinct numbers from the sequence.
Union Produces the set union of two sequences. numbers1.Union(numbers2)
Gets the union of two sequences.
Intersect Produces the set intersection of two sequences. numbers1.Intersect(numbers2)
Gets the intersection of two sequences.
Except Produces the set difference of two sequences. numbers1.Except(numbers2)
Gets the difference of two sequences.

Aggregation

Method Description Example
Aggregate Performs a custom aggregation operation on the values in a sequence. numbers.Aggregate((acc, n) => acc + n)
Calculates the sum of numbers.
Count Returns the number of elements in a sequence. numbers.Count()
Gets the count of elements in the sequence.
Sum Computes the sum of the sequence of numeric values. numbers.Sum()
Calculates the sum of numbers.
Average Computes the average of the sequence of numeric values. numbers.Average()
Calculates the average of numbers.
Min Returns the minimum value in a sequence. numbers.Min()
Gets the minimum value in the sequence.
Max Returns the maximum value in a sequence. numbers.Max()
Gets the maximum value in the sequence.

Quantifiers

Method Description Example
Any Determines whether any element of a sequence satisfies a condition. numbers.Any(n => n % 2 == 0)
Checks if any element in the sequence is even.
All Determines whether all elements of a sequence satisfy a condition. numbers.All(n => n > 0)
Checks if all elements in the sequence are positive.

Element Operators

Method Description Example
First Returns the first element of a sequence. numbers.First()
Gets the first element of the sequence.
FirstOrDefault Returns the first element of a sequence, or a default value if no such element is found. numbers.FirstOrDefault()
Gets the first element of the sequence or a default value if the sequence is empty.
Last Returns the last element of a sequence. numbers.Last()
Gets the last element of the sequence.
LastOrDefault Returns the last element of a sequence, or a default value if no such element is found. numbers.LastOrDefault()
Gets the last element of the sequence or a default value if the sequence is empty.
Single Returns the only element of a sequence that satisfies a specified condition, and throws an exception if more than one such element exists. numbers.Single(n => n == 5)
Gets the single element of the sequence that equals 5.
SingleOrDefault Returns the only element of a sequence that satisfies a specified condition or a default value if no such element exists. numbers.SingleOrDefault(n => n == 5)
Gets the single element of the sequence that equals 5 or a default value if the sequence is empty.

Element Retrieval

Method Description Example
ElementAt Returns the element at a specified index in a sequence. numbers.ElementAt(3)
Gets the element at index 3 in the sequence.
ElementAtOrDefault Returns the element at a specified index in a sequence or a default value if the index is out of range. numbers.ElementAtOrDefault(10)
Gets the element at index 10 in the sequence or a default value if the index is out of range.
First Returns the first element of a sequence that satisfies a specified condition. numbers.First(n => n % 2 == 0)
Gets the first even element in the sequence.
FirstOrDefault Returns the first element of a sequence that satisfies a specified condition or a default value if no such element is found. numbers.FirstOrDefault(n => n % 2 == 0)
Gets the first even element in the sequence or a default value if the sequence is empty.
Last Returns the last element of a sequence that satisfies a specified condition. numbers.Last(n => n % 2 == 0)
Gets the last even element in the sequence.
LastOrDefault Returns the last element of a sequence that satisfies a specified condition or a default value if no such element is found. numbers.LastOrDefault(n => n % 2 == 0)
Gets the last even element in the sequence or a default value if the sequence is empty.
